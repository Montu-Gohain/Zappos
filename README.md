# Project name : Zappos (A electronic Ecommerce App)

### Tech Stack :

- Reactjs
- Context Api
- Fetching product from an External API
- Search, and Filter option for products.
- Fully Responsive.
- Installable PWA.

<!-- How to run it locally. -->

## Run it locally

```bash
    git clone https://github.com/Montu-Gohain/Zappos.git
    npm install
    npm run start
```
